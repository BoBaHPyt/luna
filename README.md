# Luna API

REST API для справочника Организаций

## Стек

- FastAPI
- Pydantic
- SQLAlchemy
- Alembic
- SQLite ()

## Запуск

```bash
docker-compose up --build
```

## API Key

Все запросы должны содержать заголовок `X-API-Key: secret-api-key` (работает авторизация в swagger)

## Документация

- Swagger UI: http://localhost:8000/docs

## Проблемы

- Фильтрация объектов при поиске по радиусу происходит на стороне python (сначала фильтруем объекты по квадратной области на стороне бд, затем по радиусу окружности на стороне python).
- Отсутствует логирование, но тут и логировать особо нечего
- Ну и mypy будет мягко говоря очень сильно недоволен :)
